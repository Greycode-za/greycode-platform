# Greycode IoT Platform for PlatformIO

This repository contains the official **Greycode PlatformIO Platform**, designed for **Greycode IoT development boards** based on the **ESP32 architecture**.

---

## ⚙️ Installation

You can install this custom platform directly from GitHub using PlatformIO CLI:

```bash
pio platform install https://github.com/Greycode-za/greycode-platform.git
